Note: Run solr server before queryUI.py

To access the solr server:
1. Unzip the file and put in Downloads
2. go to cmd prompt and cd Downloads/solr-9.5.0/solr-9.5.0/bin
3.Run "solr start" in cmd
4. Open website link:
http://hostname:8983/solr/

To run queryUI.py:
1. Download queryIU.py and put in Downloads
2. go to cmd prompt and cd Downloads
3. Run "streamlit run queryUI.py" in cmd
4. Open website link:
http://localhost:8501/
